# 📩 Spam Detection Web App

A Machine Learning-based Spam Detection system trained on 200k+ real-world messages and deployed as a live web application using Streamlit.

---

## 🚀 Live Demo
🔗 https://spam-detection-app-ycqpuifrgwx3hyolmsvvws.streamlit.app  

---

## 📌 Project Overview

This project classifies SMS/text messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques and Machine Learning.

The goal was to build an end-to-end ML system:
- Data preprocessing
- Feature extraction
- Model training
- Model evaluation
- Web deployment

---

## 🧠 Dataset

- Combined multiple spam/ham datasets
- Total messages: 200,000+
- Balanced classes for better generalization

---

## ⚙️ Tech Stack

- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit
- Joblib
- GitHub (version control & deployment workflow)

---

## 📊 Model Performance

- Accuracy: ~95%
- Confusion Matrix evaluated
- Confidence scores shown for predictions
- Tested on real user inputs including edge cases

Special focus was given to reducing **False Positives** (important in spam detection systems).

---

## 🖥️ How It Works

1. User enters a message
2. Text is preprocessed
3. Transformed using vectorization
4. Model predicts Spam or Ham
5. Confidence score is displayed
